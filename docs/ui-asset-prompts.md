# UI Asset Prompts (Icons + Images)

Use these prompts to generate a consistent visual library for:
- FCL Import Clearance
- FTL Export - Warehouse Operations
- Import Transfer of Ownership

## 1) Global Prompt - Icon Style

Use this prefix before every icon prompt:

```text
Create a clean SaaS logistics icon in a unified style: rounded monoline vector, 2px stroke, minimal geometric details, soft corners, no text, no watermark, transparent background, centered, balanced whitespace, export as SVG and 1024x1024 PNG. Primary color #0F172A. Optional accent duotone: #0EA5E9 and #10B981. Keep icon legible at 20px.
```

## 2) Global Prompt - Illustration Style

Use this prefix before every image prompt:

```text
Create a modern logistics dashboard illustration for enterprise web UI. Flat + soft gradient style, crisp vector look, subtle depth, clean composition, no text, no watermark. Color system: slate neutrals with accents #0EA5E9, #10B981, #F59E0B, #EF4444. Leave negative space for UI overlays. Export 16:9 and 4:3.
```

## 9) Export Specs

For icons:
- SVG (primary)
- PNG: 1024x1024
- Optional sizes: 20, 24, 32
- Transparent background

For illustrations:
- PNG/WebP
- 1600x900 (16:9), 1200x900 (4:3)
- Keep a safe empty area on one side for text overlays
